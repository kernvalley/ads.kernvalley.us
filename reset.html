Web App Reset

This page is exclusively for clearing data saved by this web app. If you were instructed to navigate to this page, it is most likely because your browser was storing data that resulted in errors that could not be resolved without clearing that data.

If supported, simply by visiting this page your browser has already cleared:

  • Cookies
  • Local & Session Storage
  • HTTP Cache
  • Any Registered Service Workers
  • Service Worker Cache
  • IndexedDB databases

Note: If you are using a browser that supports the Clear-Site-Data HTTP header, you do not need to take any further action. As of Oct. 2020, this is supported in the latest versions of:

  • Google Chrome 61+
  • Mozilla Firefox 63+
  • Microsoft Edge 79+ (based on Chromium)
It is not supported in:
  • Safari (any version)
  • Internet Explorer (all versions)
  • Chrome < 61 (Sep. 2017)
  • Firefox < 63 (Sep. 2018/)
  • Edge < 79 (Jan. 2020)

If you are using an unsupported browser or you'd simply like to be extra sure data is cleared, you may manually clear some data by clicking the button below.
